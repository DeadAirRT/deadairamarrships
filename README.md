# DeadAir Amarr Ships

## Adds 7 L ships
- Abaddon
- Armageddon
- Harbinger
- Legion
- Omen
- Oracle
- Providence
## Adds 2 XL ships
- Archon
- Avatar
## Required Mods
- DeadAir Faction Damarr
- DeadAir Factions VRO
## Installation
- folder must be named "deadairamarrships"
- folder must be in game/extensions folder, not documents
